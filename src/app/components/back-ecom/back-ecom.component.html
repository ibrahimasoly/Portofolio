Backend E-commerce micro-service

Backend e-commerce en architecture microservices, réalisé en Java avec le framework Spring Boot, intégrant toutes les dépendances nécessaires pour un microservice robuste. Les outils utilisés incluent Eureka Discovery, Config Server, OpenFeign, RestTemplate, Kafka et MailDev etc. Le projet repose sur deux bases de données, relationnelle (MySQL) et non relationnelle (MongoDB), illustrant les avantages de l'architecture microservices, où chaque service peut disposer de sa propre base de données. La communication entre microservices est à la fois synchrone via REST API et asynchrone via Kafka. Le système permet également l'intégration avec d'autres API en utilisant des protocoles REST et Kafka, avec OpenFeign et RestTemplate pour les appels REST, Kafka pour la messagerie, et Resilience4j pour gérer la tolérance aux pannes d'un microservice.
